#### **Част 1: Стандартни форм-фактори (15 мин)**

**ATX Standard (PS/2):**
```
Размери: 150 × 86 × 140 mm (W × H × D)
Мощност: 300W - 2000W+
Приложение: Стандартни desktop системи
Монтаж: Винтове отзад на кутията
Вентилатор: 120mm или 140mm
```

**SFX (Small Form Factor):**
```
Размери: 125 × 63.5 × 100 mm
Мощност: 300W - 850W
Приложение: Mini-ITX системи
Вентилатор: 80mm или 92mm
Особености: SFX-L варинт (125 × 63.5 × 130 mm)
```

**TFX (Thin Form Factor):**
```
Размери: 85 × 65 × 175 mm
Мощност: 180W - 400W
Приложение: Slim desktop системи
Вентилатор: 80mm
Особености: За хоризонтални кутии
```

**Flex ATX:**
```
Размери: 81.5 × 40.5 × 150 mm
Мощност: 150W - 500W
Приложение: Ultra-compact системи
Вентилатор: 40mm (шумен)
Особености: 1U rack системи
```

#### **Част 2: Модулни конструкции (12 мин)**

**Типове по модулност:**

**1. Немодулни (Fixed cables):**
```
Предимства:
✓ По-ниска цена
✓ Надеждни връзки
✓ Няма загуби от конектори

Недостатъци:
✗ Излишни кабели
✗ Труден кабелен мениджмънт
✗ Заема повече място
```

**2. Полу-модулни (Semi-modular):**
```
Фиксирани кабели:
- 24-pin ATX
- 4+4-pin CPU

Модулни кабели:
- PCIe
- SATA
- Molex

Баланс между цена и удобство
```

**3. Напълно модулни (Fully modular):**
```
Предимства:
✓ Отличен кабелен мениджмънт
✓ Използват се само нужните кабели
✓ Лесна подмяна на кабели
✓ Custom кабели опция

Недостатъци:
✗ По-висока цена
✗ Възможност за загуба на кабели
✗ Допълнително съпротивление
```

**Конектори за модулни кабели:**
- 6-pin за периферия
- 8-pin за PCIe
- Proprietary конектори
- **ВНИМАНИЕ:** Не са универсални между марки!

**Практическа демонстрация:**
- Свързване/разкачане на модулни кабели
- Показване на различни типове конектори
- Организиране на кабелите

#### **Част 3: Системи за охлаждане (8 мин)**

**Конфигурации на вентилатори:**

**1. Bottom-mounted (вентилатор отдолу):**
```
Посока: Издухва навън
Предимства: Директно охлаждане
Проблем: Събира прах от дъното
Решение: Dust filter
```

**2. Top-mounted (вентилатор отгоре):**
```
ATX стандарт позиция
Всмуква от вътрешността
Охлажда се от case airflow
По-тихо от bottom-mount
```

**3. Passive/Fanless:**
```
0 dB операция
Големи радиатори
Ограничена мощност (<500W)
За silent PC builds
```

**4. Hybrid mode:**
```
Вентилатор спира под 40% товар
Eco mode функция
Удължен живот на вентилатора
```

**Размери на вентилатори и характеристики:**

#### **Част 4: Специални типове захранвания (7 мин)**

**Redundant захранвания:**
```
Характеристики:
- 2+ захранвания в паралел
- Hot-swap възможност
- N+1 резервиране
- Load balancing
- Server приложения

Пример: 2×750W = 1500W total
При отказ: 1×750W продължава
```

**Mining захранвания:**
```
Особености:
- Множество PCIe конектори (6-12+)
- Висока ефективност на +12V rail
- Server grade компоненти
- 2000W+ мощност
- Breakout board опции
```

**DC-DC захранвания:**
```
PicoPSU: 
- 12-24V DC вход
- 60-200W мощност
- За embedded системи
- Размер: като RAM модул
```

**Industrial захранвания:**
```
- DIN rail монтаж
- Wide input range (85-264V AC)
- -40°C до +70°C работна температура
- Conformal coating
- MTBF > 100,000 часа
```

### ПРИЛОЖЕНИЕ 2: КАЛКУЛАТОР ЗА ИЗБОР

**Формула за определяне на мощност:**
```python
# Базова формула
Required_PSU = (TDP_sum × 1.5) / Efficiency

# Пример:
components = {
    'CPU': 125,      # Ryzen 7
    'GPU': 250,      # RTX 4070
    'MB': 50,        # Chipset + VRM
    'RAM': 15,       # 32GB DDR5
    'Storage': 15,   # 2×SSD + HDD
    'Fans': 20,      # 6×fans
    'RGB': 10        # Optional
}

total_tdp = sum(components.values())  # 485W
peak_load = total_tdp × 1.3           # 630W
recommended = peak_load / 0.8          # 788W
# Избор: 850W PSU
```
