## 5. ХОД НА УРОКА

### **1. ОРГАНИЗАЦИОНЕН МОМЕНТ (5 минути)**

**Дейност на учителя:**
- Проверка на присъствието
- Разпределение по работни станции
- **КРИТИЧЕН** инструктаж за безопасност

**Правила за безопасна работа:**
```
⚠️ ЗАДЪЛЖИТЕЛНИ ПРАВИЛА:
1. ВИНАГИ използвайте антистатична защита
2. НИКОГА не докосвайте pins/контакти
3. НЕ прилагайте излишна сила
4. Проверявайте ориентацията преди поставяне
5. Термопастата НЕ е проводима (обикновено)
6. Пазете компонентите от падане
```

**ESD защита:**
- Антистатична гривна → заземяване
- Докосвайте метална част на кутията
- Работете на антистатична подложка
- Избягвайте синтетични дрехи

### **2. АКТУАЛИЗАЦИЯ НА ЗНАНИЯТА (8 минути)**

**Дейност на учителя:**
- Преговор на CPU характеристики

**Въпроси за преговор:**

2. Какви са нормалните работни температури?

### **3. ПРАКТИЧЕСКА РАБОТА (65 минути)**

#### **Част 1: CPU сокети и инсталация (20 мин)**

**Intel LGA (Land Grid Array):**
```
LGA 1700 процедура (12/13th gen):
1. Отворете socket cover:
   - Освободете retention arm
   - Вдигнете метален капак
   - НЕ махайте пластмасовия предпазител още

2. Ориентирайте процесора:
   - Намерете gold triangles (2 ъгъла)
   - Съвместете с маркировките на сокета
   - Alignment notches отстрани

3. Поставете процесора:
   - Дръжте само по ръбовете
   - Нежно положете (NO FORCE!)
   - Трябва да "седне" перфектно

4. Затворете сокета:
   - Спуснете метален капак
   - Натиснете retention arm
   - Пластмасовият капак ще изскочи
   - ЗАПАЗЕТЕ капака!

Внимание: ILM (Independent Loading Mechanism)
може да изкриви CPU - не презатягайте!
```

**AMD AM4/AM5 (PGA/LGA):**
```
AM4 (PGA) процедура:
1. Вдигнете retention arm до 90°
2. Ориентация - gold triangle
3. Процесорът "пада" в сокета
4. БЕЗ натиск - само гравитация
5. Затворете retention arm

AM5 (LGA) процедура:
- Подобна на Intel LGA
- БЕЗ метален капак
- По-деликатен механизъм
```

**Възможни грешки:**
```
❌ Огънати пинове (PGA)
   Решение: Внимателно изправяне с химикал

❌ Огънати socket pins (LGA)
   Решение: Много трудно, често fatal

❌ Грешна ориентация
   Решение: НИКОГА не форсирайте!

❌ Термопаста в сокета
   Решение: Почистване с алкохол и четка
```

#### **Част 2: Термопаста - методи и приложение (15 мин)**

**Видове термопаста:**
```
По състав:
1. Ceramic-based (бяла/сива)
   - Непроводима, безопасна
   - 3-8 W/mK проводимост
   - Пример: Arctic MX-4

2. Metal-based (сива/сребриста)
   - Малко по-добра
   - 8-12 W/mK
   - Внимание с разливане
   - Пример: Arctic Silver 5

3. Liquid metal (течен метал)
   - Най-добра: 70+ W/mK
   - ПРОВОДИМА - опасна!
   - Само за опитни
   - Пример: Thermal Grizzly Conductonaut

4. Thermal pads/putty
   - За VRAM, VRM
   - Различни дебелини
   - По-лоши от паста за CPU
```

**Методи за нанасяне:**
```
1. "Pea method" (грахово зърно):
   ✓ Най-популярен
   • Точка в центъра ~3-4mm
   [Диаграма: ·]

2. "X method":
   ✓ За големи CPU/GPU
   • Два диагонала
   [Диаграма: X]

3. "Line method":
   ✓ За правоъгълни dies
   • Тънка линия по средата
   [Диаграма: —]

4. "Spread method":
   ✓ Гарантирано покритие
   • Разнасяне с шпатула
   ✗ Въздушни мехури риск

5. "Buttered toast":
   • Тънък слой на CPU И cooler
   ✓ За direct-die cooling

Количество: "По-малко е повече!"
Оптимално: 0.2-0.5g за CPU
```

**Практическа демонстрация:**
1. Почистване на стара паста:
   - Изопропилов алкохол 99%
   - Микрофибърна кърпа
   - Кръгови движения от център
   - До огледален блясък

2. Нанасяне на нова:
   - Малко количество
   - Един опит - не поправяйте
   - Натискът ще я разпредели

#### **Част 3: Монтаж на охладители (20 мин)**

**Stock (Box) охладители:**

**Intel Stock cooler:**
```
Push-pin система:
1. Ориентирайте със стрелките навън
2. Подравнете с отворите
3. Натиснете диагонално противоположни
4. Чува се "click"
5. Проверете отдолу за фиксиране

Премахване:
- Завъртете 90° обратно на часовника
- Издърпайте нагоре
```

**AMD Stock (Wraith) cooler:**
```
Clip система (AM4):
1. Премахнете пластмасови скоби
2. Поставете cooler
3. Закачете клиповете
4. Фиксирайте с лоста

Винтова система (AM5):
1. Използва същия backplate
2. 4 винта в ъглите
3. Кръстосано затягане
```

**Aftermarket въздушни охладители:**

**Tower cooler монтаж:**
```
Универсална процедура:
1. Backplate монтаж:
   - Отдолу на MB
   - Специфичен за socket
   - Винтове/стойки през отворите

2. Mounting brackets:
   - Върху процесора
   - Съобразени с ориентацията

3. Heatsink монтаж:
   - RAM clearance проверка!
   - Ориентация към exhaust
   - Притягане на винтове/скоби

4. Вентилатор:
   - Посока на издухване
   - Свързване към CPU_FAN header

Съвет: Монтирайте RAM първо!
```

**AIO водно охлаждане:**
```
240/280/360mm radiator:

1. Планиране:
   - Top mount (exhaust) - топъл въздух навън
   - Front mount (intake) - хладен въздух, топли компоненти

2. Радиатор:
   - Първо вентилатори към радиатор
   - После радиатор към кутия
   - Тръбите към долната част (ако възможно)

3. Помпа/блок:
   - Същият mounting като tower
   - Ориентация на тръбите
   - USB/SATA захранване за RGB/контрол

4. Кабели:
   - Pump към AIO_PUMP/CPU_FAN
   - Вентилатори към CPU_FAN/CHA_FAN
   - RGB към съответните headers

Важно: Помпата НЕ трябва да е най-високата точка!
```

#### **Част 4: Проверка и тестване (10 мин)**

**POST тест и проверки:**
```
1. Визуална инспекция:
   - Cooler добре фиксиран?
   - Вентилатор свързан?
   - Няма кабели в перките?

2. Първо включване:
   - Наблюдавайте вентилатора
   - Влезте в BIOS веднага
   - Проверете CPU температура
   - Idle: 30-45°C нормално

3. Stress test:
   - Prime95 Small FFTs
   - Наблюдавайте temps
   - Max safe: 85-95°C (зависи от CPU)
   - Throttling? = лош контакт

4. Проблеми и решения:
   - Високи temps = re-mount
   - Неравномерни core temps = лош контакт
   - Pump не работи = проверете захранване
```

### ПРИЛОЖЕНИЕ 2: ТЕМПЕРАТУРНИ НОРМИ

**Safe Operating Temperatures:**
```
Intel:
- Idle: 30-45°C
- Gaming: 60-80°C
- Stress: 80-95°C
- Tjmax: 100-105°C

AMD:
- Idle: 35-50°C
- Gaming: 65-85°C
- Stress: 85-90°C
- Tjmax: 90-95°C

Ambient влияние:
Delta = CPU temp - Room temp
Good cooling: Delta < 30°C idle
                    < 50°C load
```

### ПРИЛОЖЕНИЕ 3: TROUBLESHOOTING

**Чести проблеми и решения:**
```
Проблем: Много високи температури
Причини и решения:
1. Лош контакт
   → Re-mount охладителя
2. Стара/малко термопаста
   → Почистете и re-apply
3. Забравен стикер на охладителя
   → Махнете защитното фолио!
4. Вентилатор не върти
   → Проверете захранването
5. Запушен радиатор
   → Почистете от прах

Проблем: Неравномерни core temps
(>10°C разлика)
Причини:
1. Неравномерен mounting pressure
2. Изкривен IHS (Intel)
3. Лоша паста разпределение

Проблем: System не стартира
След CPU работа:
1. CPU power незакачен (4+4/8 pin)
2. RAM излязъл при работа
3. Огънати пинове
4. ESD повреда (рядко)
```
