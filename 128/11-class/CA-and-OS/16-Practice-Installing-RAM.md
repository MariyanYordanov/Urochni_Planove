### **2. ДЕМОНСТРАЦИЯ (15 минути)**

#### **Част 1: Подготовка (5 мин)**

**Стъпки преди монтаж:**
```
1. Изключете компютъра (shutdown, не sleep)
2. Изключете захранващия кабел
3. Натиснете Power бутон (разрежда остатъчния ток)
4. Свържете антистатичен накитник към корпуса
5. Поставете компютъра на антистатична постелка
6. Отворете страничния панел
```

**Идентификация на слотове:**
```
Типична 4-DIMM платка (Intel/AMD):

Процесор       DIMM слотове
   █           A1  A2  B1  B2
   █           │   │   │   │
               1   2   3   4

За Dual Channel с 2 модула:
┌───┬───┬───┬───┐
│ 1 │ 2 │ 3 │ 4 │
└───┴───┴───┴───┘
      ▲       ▲
      A2      B2  ← Инсталирайте тук!

Цветово кодиране (много производители):
- Един цвят = един канал
- A2 и B2 обикновено са препоръчителни за 2 модула
```

#### **Част 2: Инсталация (5 мин)**

**Стъпка по стъпка:**
```
1. ОТВОРЕТЕ КЛИПСОВЕТЕ
   - Натиснете клипсовете в края на слота навън
   - Някои платки имат клипс само от едната страна
   
2. ОРИЕНТИРАЙТЕ МОДУЛА
   - Намерете ключа (notch) на модула
   - Подравнете с прореза в слота
   - DDR4/DDR5 ключът е изместен от центъра!
   
3. ПОСТАВЕТЕ МОДУЛА
   - Поставете под лек ъгъл (около 15°)
   - Подравнете и двата края едновременно
   - НЕ натискайте силно!
   
4. НАТИСНЕТЕ ВЕРТИКАЛНО
   - Равномерен натиск върху двата края
   - Чуйте "щракване" от клипсовете
   - Клипсовете трябва да се затворят автоматично
   
5. ПРОВЕРЕТЕ
   - И двата клипса затворени?
   - Модулът седи равномерно?
   - Няма видим ъгъл?
```

**Демонстрация на грешки:**
```
❌ ГРЕШКИ ДА СЕ ИЗБЯГВАТ:
- Обърнат модул (ключът не съвпада)
- Само един край натиснат
- Прекалена сила
- Докосване на контактите
- Работа без ESD защита
```

#### **Част 3: Демонтаж (5 мин)**

**Стъпки за изваждане:**
```
1. Натиснете клипсовете навън едновременно
2. Модулът ще "изскочи" леко нагоре
3. Хванете за краищата и извадете
4. Поставете в антистатична торбичка
```

### **3. ПРАКТИЧЕСКА РАБОТА (40 минути)**

#### **Задача 1: Инсталация на RAM (15 мин)**

**Инструкции за учениците:**

```
РАБОТЕН ЛИСТ - ИНСТАЛАЦИЯ НА RAM

Група: _____________  Компютър: _____________

ПРЕДИ НАЧАЛОТО:
□ Изключен компютър
□ Изключен захранващ кабел
□ Антистатичен накитник свързан
□ Идентифицирани правилните слотове

ИНСТАЛАЦИЯ:
□ Отворени клипсове
□ Модул правилно ориентиран
□ Модул поставен равномерно
□ Чуто щракване
□ Клипсове затворени

ПРОВЕРКА:
□ Визуална инспекция OK
□ Захранване свързано
□ Компютър стартира
□ BIOS разпознава паметта

ДАННИ:
- Общо RAM преди: _______ GB
- Инсталиран модул: _______ GB
- Общо RAM след: _______ GB
- Двата канала активни: □ Да □ Не
```

**Ученици по двойки:**
- Ученик 1 инсталира, Ученик 2 наблюдава и проверява
- Размяна на ролите

#### **Задача 2: Тестване на памет (15 мин)**

**Windows Memory Diagnostic:**
```
Стъпки:
1. Натиснете Win + R
2. Въведете: mdsched.exe
3. Изберете "Restart now and check for problems"
4. Компютърът рестартира и стартира тест
5. Изчакайте завършване (5-10 минути базов тест)

Резултати:
- Ако няма грешки: паметта е OK
- Ако има грешки: записва в Event Log
```

**Проверка на резултати:**
```
След рестартиране:
1. Event Viewer → Windows Logs → System
2. Потърсете "MemoryDiagnostics-Results"
3. Прочетете съобщението

Или през CMD:
wevtutil qe System /c:1 /f:text /q:"*[System[Provider[@Name='MemoryDiagnostics-Results']]]"
```

**CPU-Z проверка:**
```
Стъпки:
1. Стартирайте CPU-Z
2. Memory tab: проверете Channel # (Dual?)
3. SPD tab: проверете всеки слот
4. Запишете данните в работния лист
```

#### **Задача 3: XMP активация (10 мин)**

**Инструкции:**
```
1. Рестартирайте и влезте в BIOS/UEFI (DEL или F2)
2. Намерете раздел за памет:
   - ASUS: Ai Tweaker → Ai Overclock Tuner
   - MSI: OC → A-XMP
   - Gigabyte: M.I.T. → Extreme Memory Profile
   - ASRock: OC Tweaker → DRAM Configuration

3. Променете от "Auto" на "XMP Profile 1" (или "XMP I")

4. Save & Exit (F10)

5. След рестартиране, проверете в CPU-Z:
   - DRAM Frequency = очаквана стойност / 2
   - Пример: DDR4-3200 → трябва да видите 1600 MHz
```

**Работен лист:**
```
XMP АКТИВАЦИЯ

Преди XMP:
- DRAM Frequency: _______ MHz
- DDR Effective: _______ MT/s
- Timings: CL___ - ___-___-___

След XMP:
- DRAM Frequency: _______ MHz  
- DDR Effective: _______ MT/s
- Timings: CL___ - ___-___-___
- XMP Profile: _______________

Разлика в честотата: +_______ MHz
```

### **4. TROUBLESHOOTING СЦЕНАРИИ (15 минути)**

**Казус 1: Компютърът не стартира след инсталация**
```
Симптоми: Черен екран, без POST beeps

Checklist:
□ Модулът е напълно поставен?
□ Клипсовете са затворени?
□ Правилни ли са слотовете? (A2, B2)
□ Модулът е съвместим? (DDR4 vs DDR5)

Решение:
1. Извадете модула, преинсталирайте
2. Опитайте друг слот
3. Опитайте само с един модул
4. Проверете QVL листа на производителя
```

**Казус 2: Само половината RAM се вижда**
```
Симптоми: Инсталирани 16GB, виждат се 8GB

Възможни причини:
- Модул не е разпознат
- Дефектен модул
- Дефектен слот
- BIOS ограничение

Решение:
1. CPU-Z → SPD tab → проверете всеки слот
2. Разменете модулите между слотовете
3. Тествайте всеки модул поотделно
```

**Казус 3: RAM показва по-ниска честота**
```
Симптоми: DDR4-3200 показва 2133 MHz

Причина: XMP не е активиран (JEDEC default)

Решение:
1. Влезте в BIOS
2. Активирайте XMP профил
3. Save & Exit
4. Проверете в CPU-Z
```

**Казус 4: Blue Screen свързан с памет**
```
BSOD кодове свързани с RAM:
- MEMORY_MANAGEMENT
- PAGE_FAULT_IN_NONPAGED_AREA
- KERNEL_DATA_INPAGE_ERROR
- BAD_POOL_HEADER

Действия:
1. Стартирайте MemTest86 (пълен тест)
2. Ако има грешки → идентифицирайте дефектния модул
3. Тествайте всеки модул поотделно
4. Проверете XMP настройките (намалете ако е нужно)
```

## 8. ПРИЛОЖЕНИЯ

### ПРИЛОЖЕНИЕ 1: DUAL CHANNEL СПРАВОЧНИК

**ASUS/Gigabyte (повечето модели):**
```
За 2 модула: Слотове A2 и B2 (обикновено 2 и 4)
┌───┬───┬───┬───┐
│ 1 │ 2 │ 3 │ 4 │  (номерация от CPU)
│   │ █ │   │ █ │  ← Правилно
└───┴───┴───┴───┘
```

**MSI:**
```
Слотове DIMMA2 и DIMMB2
Проверете manual - може да варира
```

**Процесори с 2 DIMM слота:**
```
┌───┬───┐
│ 1 │ 2 │
│ █ │ █ │  ← И двата за Dual Channel
└───┴───┘
```

### ПРИЛОЖЕНИЕ 2: MEMTEST86 РЪКОВОДСТВО

**Създаване на bootable USB:**
```
1. Изтеглете от memtest86.com
2. Стартирайте imageUSB.exe (включено)
3. Изберете USB устройство
4. "Write image to USB drive"
```

**Интерпретация на резултати:**
```
PASS = тестът е минал успешно
FAIL = открити грешки

Грешки (Errors):
- 0 грешки = памет OK
- Всяка грешка = потенциален проблем

При грешки:
1. Тествайте модулите поотделно
2. Тествайте в различни слотове
3. Дефектният модул = замяна
```

### ПРИЛОЖЕНИЕ 3: BIOS НАСТРОЙКИ ПО ПРОИЗВОДИТЕЛ

**ASUS:**
```
Ai Tweaker → Ai Overclock Tuner → XMP I / XMP II
или
EZ Mode → XMP button (ако има)
```

**MSI:**
```
OC → A-XMP → Profile 1 / Profile 2
или
EZ Mode → A-XMP switch
```

**Gigabyte:**
```
M.I.T. → Advanced Memory Settings → Extreme Memory Profile (XMP)
или
Easy Mode → XMP
```

**ASRock:**
```
OC Tweaker → DRAM Configuration → Load XMP Setting
```
