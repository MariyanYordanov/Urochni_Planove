### ПРИЛОЖЕНИЕ 3: ОТГОВОРИ НА ТЕСТА

```
КЛЮЧ С ОТГОВОРИ - САМО ЗА УЧИТЕЛЯ

РАЗДЕЛ А:
1-b, 2-d, 3-c, 4-c, 5-b, 6-c, 7-c, 8-b, 9-c, 10-c,
11-b, 12-b, 13-b, 14-b, 15-c

РАЗДЕЛ Б:
1-Г, 2-В, 3-В, 4-В, 5-Г, 6-В, 7-Г, 8-Г, 9-В, 10-В

РАЗДЕЛ В:
1-c, 2-d, 3-a, 4-b, 5-e, 6-g, 7-f, 8-i, 9-h, 10-j

РАЗДЕЛ Г (примерни отговори):

1. Процес vs Нишка:
   - Процес: независима програма с отделно адресно пространство
   - Нишка: лек процес, споделя ресурси с други нишки
   - Нишките са по-ефективни за комуникация

2. Виртуална памет:
   - Абстракция над физическата памет
   - Използва дисково пространство (swap)
   - Позволява изпълнение на програми по-големи от RAM

3. Йерархия на паметта:
   Регистри → L1 кеш → L2 кеш → L3 кеш → RAM → SSD → HDD

4. Функции на ОС:
   - Управление на процеси
   - Управление на паметта
   - Управление на файлова система
   - Управление на I/O устройства
   - Сигурност и защита

5. BIOS vs UEFI:
   - BIOS: 16-битов, ограничен до 2TB дискове
   - UEFI: 32/64-битов, поддържа GPT, Secure Boot
   - UEFI има графичен интерфейс
```
