**Case Study 1: APT1 - Operation Aurora (2009-2010)**

**Campaign Overview:**
Operation Aurora represented one на first publicly documented sophisticated APT campaigns, targeting Google и 30+ other companies.

**Technical Analysis:**
- **Initial Vector:** Watering hole attacks на industry websites
- **Exploitation:** Internet Explorer zero-day vulnerability (CVE-2010-0249)

- **Persistence:** Registry modification, service installation
- **C&C Communication:** Encrypted HTTP traffic to compromised websites
- **Objectives:** Gmail account access, intellectual property theft

**Attribution Evidence:**
- **Infrastructure Analysis:** Chinese hosting providers, .cn domains
- **Operational Patterns:** Beijing business hours activity
- **Linguistic Indicators:** Chinese language artifacts в code
- **Tactical Similarities:** Overlap със known PLA Unit 61398 operations

**Impact Assessment:**
- **Immediate:** Google China withdrawal, diplomatic tensions
- **Long-term:** Enhanced private sector threat awareness
- **Policy Response:** US cyber deterrence strategy development
- **Industry Change:** Security vendor APT focus increase
